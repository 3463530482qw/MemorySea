# MemorySea（记忆之海）

> 基于 C++23 的自研游戏引擎 + 五款迷你游戏合集
> 
> 从零构建 Vulkan 渲染管线 | DLL 热插拔场景系统 | 手写神经网络 AI | 过程化物理动画

---

## 🎮 游戏列表

| 游戏 | 亮点 |
|------|------|
| **俄罗斯方块** | 全部 7 种方块 × 4 旋转状态，幽灵预览，消行计分，逐级加速 |
| **贪吃蛇** | 30×22 网格，速度递增，流畅方向输入 |
| **数独** | 预设谜题，实时行/列/宫冲突标红，一键清除 |
| **平台跳跃** | 弹簧-质量水波模拟（80 段），溅射粒子，弹性加速度曲线 |
| **主菜单** | 三次样条曲线流光动画，呼吸透明度，翻页过渡特效 |

## 🏗️ 引擎架构

```
┌─────────────────────────────────────────┐
│  main.cpp — 入口主循环                   │
├─────────────────────────────────────────┤
│  youklx/                                │
│  ├─ window/    SDL3 窗口 + 输入事件      │
│  ├─ vulkan/    Vulkan 渲染器（3 管线）    │
│  ├─ draw/      绘制命令（线/图/字）       │
│  ├─ image/     PNG 加载 + 纹理图集       │
│  ├─ font/      字形光栅化 + 多页图集      │
│  ├─ scene/     DLL 场景加载/切换         │
│  ├─ thread/    更新+绘制双线程 + 顶点工作池│
│  └─ ai/        前馈神经网络 + 反向传播    │
├─────────────────────────────────────────┤
│  sence/ — 场景 DLL（5 个游戏）           │
└─────────────────────────────────────────┘
```

- **渲染**：自研 Vulkan RAII 封装，线条/图片/字体三条独立管线，TriangleStrip/TriangleList 拓扑
- **场景系统**：每个游戏编译为独立 DLL，运行时 `LoadLibrary` 动态加载，`scene.ptr` 字符串指针实现秒切场景
- **双线程**：更新线程（游戏逻辑 + GPU 渲染）+ 绘制线程（构建绘制命令）+ 持久化工作池并行计算顶点
- **字体**：开源霞鹜文楷字体，`stb_truetype` 光栅化，多页图集竖直堆叠，按需延迟加载字形
- **坐标系统**：1600×900 逻辑分辨率，自动 letterbox 适配任意窗口

## 🚀 构建与运行

### 依赖

| 组件 | 说明 |
|------|------|
| MinGW-w64 (g++ 14+) | C++23 编译器 |
| CMake 3.28+ | 构建系统 |
| Python 3 | 构建脚本 |
| Vulkan SDK | 渲染 API |
| SDL3 | 窗口与输入 |
| stb_image / stb_truetype | 图片/字体加载 |
| inih | INI 配置解析 |
| spine-runtimes 4.3 | 骨骼动画（预留） |

### 构建

```bash
python build.py
```

脚本自动完成：着色器编译 → CMake 生成 → 编译 → 复制 DLL/资源 → 运行。

场景 DLL 独立编译（`sence/*/bulid.py`），使用 `-std=c++23 -Os -s`。

## 🎯 技术亮点

### 1. 过程化水波物理 (`sence/jump/`)
80 段弹簧-质量链模拟水面波动，含阻尼、波传播和边界消能：
```
每个质点：加速度 = (左侧拉力 + 右侧拉力 - 阻尼×速度 - 刚度×位移) / 质量
```

### 2. 自研神经网络 (`youklx/ai/`)
无第三方依赖的纯 C++ 前馈网络，支持：
- 任意隐藏层数/神经元数
- 4 种激活函数（含自研 `youk` 函数）
- 完整反向传播 + 梯度下降
- 二进制持久化（保存/加载）

### 3. DLL 场景系统 (`youklx/scene/`)
场景热切换：修改 `scene.ptr` 字符串即可切换。每个游戏独立编译、独立加载，主程序通过 `__declspec(dllexport/dllimport)` 共享全局渲染对象。

### 4. 过程化 UI 动画
三次样条插值曲线、弹性滚动（弹簧物理）、呼吸透明度、旋转动画——全部手写，不依赖任何 UI 库。
